# 💰 Personal Finance Tracker

A modern Personal Finance Tracker built with Streamlit.

## 🚀 Live Demo

https://personal-finance-tracker-d2fvxd2a6xaxhheepquvyx.streamlit.app/

---

## ✨ Features

- Add Income
- Add Expenses
- Dashboard
- Expense Pie Chart
- Search & Filter Transactions
- Edit Transactions
- Delete Transactions
- Current Balance
- CSV Data Storage
- Responsive Multi-page UI

---

## 🛠 Technologies Used

- Python
- Streamlit
- Pandas
- Plotly

---

## 📂 Project Structure

```
finance_trk.py
pages/
│
├── Dashboard
├── Add Income
├── Add Expense
└── Manage Transactions
```

---

## ▶️ Run Locally

Clone the repository

```bash
git clone https://github.com/somyakumar2007-create/Personal-finance-tracker.git
```

Install requirements

```bash
pip install -r requirements.txt
```

Run

```bash
streamlit run finance_trk.py
```
